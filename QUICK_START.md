# QUICK START GUIDE

Welcome to the WeChat Mini Program AI! This quick start guide will help you set up your project in no time. Follow the steps below:

## Step 1: Prerequisites
- Ensure you have Node.js installed on your machine. You can download it from [Node.js Official Website](https://nodejs.org/).
- Install WeChat developer tools from [WeChat DevTools](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html).

## Step 2: Clone the Repository
Open your terminal and run the following command to clone the repository:
```bash
git clone https://github.com/suyan54538-commits/wechat-mini-program-ai.git
```

## Step 3: Install Dependencies
Navigate to the project directory:
```bash
cd wechat-mini-program-ai
```
Then, install the necessary dependencies:
```bash
npm install
```

## Step 4: Start the Development Server
To start the development server, run:
```bash
npm run dev
```

## Step 5: Open WeChat Developer Tools
Open WeChat Developer Tools, and import the directory you just cloned (`wechat-mini-program-ai`).

## Step 6: Test Your Application
You can now test your application in the WeChat Developer Tools. Make sure everything works correctly!

## Step 7: Build for Production
Once you are satisfied with your application, build it for production with the following command:
```bash
npm run build
```
